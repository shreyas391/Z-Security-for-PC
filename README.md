# Z-Security-for-PC

We're going to provide Z+ security for our PC using an MFRC522 reader/writer module with Arduino Nano.

PROJECT'S AGENDA:
  - The agenda is to Turn ON our PC using one single authorized MFRC522 card.
  - When we tap the authorized MFRC card to the reader, The PC Turns ON.
  - Without the authorized card, The PC won't Turn ON.
  - By implementing this project, We provide advance security for our PC.
